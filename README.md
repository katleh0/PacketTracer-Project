# PacketTracer-Project
Designing a reasonably priced network for a one storey office.

For a new one-story office, a high-performance, reasonably priced network was created using Cisco Packet Tracer. The solution supports more than 200 devices, including IoT and guest devices, and allows for future virtual office expansion and secure remote connectivity through the use of VLAN segmentation, redundant core routing, and mixed wired/Wi-Fi access.
